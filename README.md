# Inequality Battle: ตะลุยโลกอสมการ

เกมกระดาน Streamlit สำหรับผู้เล่น 2 คน โดยเลือกเนื้อหาก่อนเริ่มเกม:

1. การแก้อสมการโดยใช้สมบัติการบวกของการไม่เท่ากัน
2. การแก้อสมการโดยใช้สมบัติการคูณของการไม่เท่ากัน
3. การแก้อสมการเชิงเส้นตัวแปรเดียว

คลังโจทย์แยกตามเนื้อหาอยู่ในโฟลเดอร์ `questions`

## เริ่มใช้งาน

```bash
pip install -r requirements.txt
streamlit run app.py
```

## โครงสร้างโจทย์

คลังโจทย์แยกเป็น 3 ไฟล์เพื่อให้ดูแลง่าย:

```text
questions
  addition.json        # 4 ระดับ
  multiplication.json  # 4 ระดับ
  linear.json          # 5 ระดับ
```

เพิ่มโจทย์ใหม่ในรายการของเนื้อหาและระดับที่ต้องการ โดยใช้รูปแบบนี้:

```json
{
  "text": "จงแก้อสมการ x + 5/4 ≤ 2",
  "latex": "x + \\frac{5}{4} \\leq 2",
  "answer": "x ≤ 3/4",
  "choices": [
    {"value": "x ≤ 3/4", "latex": "x \\leq \\frac{3}{4}"},
    {"value": "x ≥ 3/4", "latex": "x \\geq \\frac{3}{4}"}
  ]
}
```

`text` และ `value` ใช้เก็บข้อความอ่านง่ายและตรวจคำตอบ ส่วน `latex` ใช้แสดงสูตรบนหน้าจอ จึงแสดงเศษส่วนแบบซ้อนบน-ล่างได้ เช่น `\\frac{1}{3}` แทน `1/3`

แต่ละโจทย์ควรมีตัวเลือก 4 ข้อ และค่า `answer` ต้องตรงกับ `value` ของคำตอบที่ถูกต้องทุกตัวอักษร

ระบบจะปรับระดับของผู้เล่นแต่ละคนอัตโนมัติ: ตอบถูกขึ้น 1 ระดับ ตอบผิดลดลง 1 ระดับ โดยไม่ต่ำกว่าระดับ 1 และไม่เกินระดับสูงสุดของเนื้อหานั้น

เมื่อเกมจบ ระบบจะบันทึกผลลง Google Sheets หากตั้งค่า Secrets ไว้ โดยคอลัมน์สุดท้ายเป็นรายการโจทย์ที่ตอบผิด พร้อมคำตอบของผู้เล่นและเฉลย

เมื่อต้องการนำเข้าไฟล์ข้อความต้นฉบับชุดใหม่ ใช้คำสั่ง:

```bash
python tools/import_questions.py
```
